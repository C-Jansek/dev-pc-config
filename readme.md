1. Install Ansible
    ```bash
    sudo apt-get install software-properties-common
    sudo apt-add-repository ppa:ansible/ansible
    sudo apt-get update
    sudo apt-get install ansible
    ```
2. Ansible pull
    ```bash
    sudo ansible-pull -U https://github.com/C-Jansek/dev-pc-config.git
    ```
